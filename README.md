# one-cent-node-project

Project created in less than a minute.

## Description

Simple app with two endpoints. 

This mini project was created to learn how to deploy to [railway.app](https://railway.app/)

## Getting Started

### Dependencies

* express
* nodemon (dev)

### Executing program


```
npm install
node run start
```
